# 🇯🇵 NISA Tracker

Real-time portfolio tracker for Japanese NISA accounts.

**Live demo:** https://nisa-tracker-app.onrender.com

**Features:**
- Live prices from Tokyo Stock Exchange (eMAXIS Slim S&P500, All Country, USD/JPY)
- Portfolio performance calculator with gain/loss in ¥
- NISA contribution limit tracker
- ML-based risk analysis using annualized volatility
- Interactive price history charts

**Stack:** FastAPI · Python · React · Vite · Recharts · yfinance · Render
